# Web Bot Manager 🤖

A modern, responsive web application for managing and interacting with AI bots through a clean, intuitive interface. Features a comprehensive dashboard with bot management, real-time chat, analytics, and theme customization.

## ✨ Features

### 🎨 Modern UI/UX
- **Dual Theme Support**: Light & dark modes with smooth transitions
- **Responsive Design**: Fully responsive across desktop, tablet, and mobile
- **Clean Interface**: Intuitive sidebar navigation with visual hierarchy
- **Interactive Elements**: Hover effects, animations, and smooth transitions

### 🤖 Bot Management
- **Bot Dashboard**: Grid view of all bots with status indicators
- **Bot Creation**: Modal-based bot creation with configuration options
- **Bot Filtering**: Search and filter bots by name or status
- **Status Tracking**: Real-time status indicators (active/inactive)
- **Quick Actions**: Start/stop, edit, and delete bots directly from cards

### 💬 Real-time Chat Interface
- **Conversation History**: Persistent chat sessions with timestamps
- **Typing Indicators**: Visual feedback when bots are responding
- **Message Formatting**: Different styling for user vs bot messages
- **Character Counter**: Input validation with character limits
- **Chat Controls**: Clear history, export conversations

### 📊 Analytics Dashboard
- **Performance Metrics**: Key statistics (total bots, active sessions, messages, uptime)
- **Interactive Charts**: Visual data representation using Chart.js
- **Usage Trends**: Time-based analytics for bot interactions
- **Quick Stats**: At-a-glance performance indicators

### 🔧 Advanced Features
- **API Status Monitoring**: Real-time API connectivity indicators
- **Toast Notifications**: Success, error, and warning notifications
- **Modal System**: Reusable modal components for forms and confirmations
- **Form Validation**: Input validation with visual feedback
- **Accessibility**: Keyboard navigation and screen reader support
